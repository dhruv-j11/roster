# Roster - Workforce Optimization Dashboard

A full-stack Next.js application for analyzing employee data and generating workforce optimization insights.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Features

- CSV upload with employee data
- Real-time data processing and analysis
- Interactive dashboard with KPIs
- Team performance charts
- AI-powered recommendations
- Beautiful dark UI with particle animations

## CSV Format

Upload a CSV with the following columns:
- employee_id
- name
- role
- hourly_rate
- hours_worked
- output_score
- team

## Deploy to Vercel

The app is ready to deploy to Vercel:

```bash
vercel
```
